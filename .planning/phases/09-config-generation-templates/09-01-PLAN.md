- NO ports mapping (nginx proxies via /notebook/)

- volumes: [opennotebook_data:/app/data]
- depends_on: surrealdb (condition: service_healthy)
- profiles: [opennotebook]
- restart: always

**dbgpt service** (profile: dbgpt):
- image: eosphorosai/dbgpt-openai:${DBGPT_VERSION}
- command: "dbgpt start webserver --config /app/configs/dbgpt-proxy-ollama.toml"

- volumes:
  - dbgpt_data:/app/pilot/data
  - dbgpt_message:/app/pilot/message

- profiles: [dbgpt]
- restart: always
