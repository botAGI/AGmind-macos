**2. `tests/helpers/bin/lsof` (NEW):**
Mock that simulates port checking. Uses `MOCK_PORTS_IN_USE` env var formatted as `"80:nginx:1234,3000:node:5678"` (comma-separated `port:process:pid` tuples).

Parse the `-iTCP:PORT` argument to extract the port number. If that port appears in `MOCK_PORTS_IN_USE`, print a header line and a data line with the process name and PID, exit 0. If not found, exit 1.

IMPORTANT: The lsof mock MUST handle the subshell pipe issue. The research code has a bug: the `while read` loop runs in a subshell due to the pipe, so `exit 0` only exits the subshell. Instead use a different approach:

```bash
#!/bin/bash
# Mock lsof for testing port detection
# MOCK_PORTS_IN_USE="80:nginx:1234,3000:node:5678"
_port=""
for arg in "$@"; do
  case "$arg" in
    -iTCP:*) _port="${arg#-iTCP:}" ;;
  esac
done

if [ -z "$_port" ] || [ -z "${MOCK_PORTS_IN_USE:-}" ]; then
  exit 1
fi

# Use a temp approach to avoid subshell exit issue
_found=1
_old_ifs="$IFS"
IFS=","
for _entry in $MOCK_PORTS_IN_USE; do
  _eport="${_entry%%:*}"
  _rest="${_entry#*:}"
  _ename="${_rest%%:*}"
  _epid="${_rest#*:}"
  if [ "$_eport" = "$_port" ]; then
    echo "COMMAND     PID USER   FD   TYPE DEVICE SIZE/OFF NODE NAME"
    echo "${_ename}     ${_epid} user   4u  IPv4 0x0    0t0  TCP *:${_port} (LISTEN)"
    _found=0
    break
  fi
done
IFS="$_old_ifs"
exit $_found
```

**3. `tests/helpers/bin/docker` (NEW):**
Mock that simulates Docker socket connectivity. Uses `MOCK_DOCKER_SOCKET` env var: `"desktop"`, `"colima"`, `"both"`, or `"none"` (default: `"none"`).

For the `info` subcommand: check `DOCKER_HOST` env var to determine which socket is being probed. If `DOCKER_HOST` contains `docker/run` and `MOCK_DOCKER_SOCKET` is `"desktop"` or `"both"`, exit 0. If `DOCKER_HOST` contains `colima` and `MOCK_DOCKER_SOCKET` is `"colima"` or `"both"`, exit 0. Otherwise exit 1.

For any other subcommand: echo `"mock: docker $*"` and exit 0.

```bash
#!/bin/bash
# Mock docker for testing socket connectivity
# MOCK_DOCKER_SOCKET="desktop"|"colima"|"both"|"none" (default: none)
_socket="${MOCK_DOCKER_SOCKET:-none}"
case "$1" in
  info)
    if [ "$_socket" = "none" ]; then
      exit 1
    fi
    case "${DOCKER_HOST:-}" in
      *colima*)
        [ "$_socket" = "colima" ] || [ "$_socket" = "both" ] && exit 0 || exit 1
        ;;
      *docker/run*|*docker.sock*)
        [ "$_socket" = "desktop" ] || [ "$_socket" = "both" ] && exit 0 || exit 1
        ;;
      *)
        # No DOCKER_HOST set -- if any socket available, succeed
        [ "$_socket" != "none" ] && exit 0 || exit 1
        ;;
    esac
    ;;
  *)
    echo "mock: docker $*"
    exit 0
    ;;
esac
```
