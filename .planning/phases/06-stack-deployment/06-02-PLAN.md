**scripts/backup.sh** (~40 lines):
- Shebang: `#!/bin/bash`
- `set -euo pipefail`
- `AGMIND_DIR="${AGMIND_DIR:-/opt/agmind}"`
- `BACKUP_BASE="${HOME}/Library/Application Support/AGMind/backups"`
- `_ts() { date "+%Y-%m-%d_%H%M%S"; }`
- Create backup dir: `mkdir -p "$BACKUP_BASE"`
- Create timestamped backup: `BACKUP_DIR="${BACKUP_BASE}/agmind-$(_ts)"`
- `mkdir -p "$BACKUP_DIR"`
- Copy config files: `cp "${AGMIND_DIR}/.env" "${BACKUP_DIR}/" 2>/dev/null || true`
- `cp "${AGMIND_DIR}/credentials.txt" "${BACKUP_DIR}/" 2>/dev/null || true`
- `cp "${AGMIND_DIR}/docker-compose.yml" "${BACKUP_DIR}/" 2>/dev/null || true`
- `cp "${AGMIND_DIR}/nginx.conf" "${BACKUP_DIR}/" 2>/dev/null || true`
- `cp "${AGMIND_DIR}/versions.env" "${BACKUP_DIR}/" 2>/dev/null || true`
- Log result: `echo "[$(_ts)] Backup completed: ${BACKUP_DIR}" >> "${AGMIND_DIR}/logs/backup.log"`
- Prune old backups: keep last 7, remove older ones:
  ```bash
  # Prune old backups (keep last 7)
  local count
  count=$(ls -1d "${BACKUP_BASE}"/agmind-* 2>/dev/null | wc -l | tr -d ' ')
  if [ "$count" -gt 7 ]; then
      ls -1d "${BACKUP_BASE}"/agmind-* | sort | head -n "$((count - 7))" | while read -r old; do
          rm -rf "$old"
      done
  fi
  ```
  Note: since this is a standalone script (not a function), use a variable instead of `local`.

Create **lib/backup.sh** (~80 lines):

- File header matching existing lib/*.sh style: "AGMind LaunchAgent management, Sourced by install.sh for Phase 9 (Complete)"
- `set -eEuo pipefail`
- Depends on: lib/common.sh (log_info, log_warn, die, ensure_directory, AGMIND_DIR)

Internal function `_load_launch_agent()`:
- Arg: `$1` = full path to plist file
- Extract label: `label=$(basename "$1" .plist)`
- Idempotent check: `if launchctl list "$label" >/dev/null 2>&1; then log_info "LaunchAgent ${label} already loaded -- skipping"; return 0; fi`
- Get user ID: `uid=$(id -u)`
- Try modern bootstrap: `if launchctl bootstrap "gui/${uid}" "$1" 2>/dev/null; then log_info "LaunchAgent ${label} loaded (bootstrap)"; return 0; fi`
- Fallback to legacy: `if launchctl load "$1" 2>/dev/null; then log_info "LaunchAgent ${label} loaded (legacy)"; return 0; fi`
- Both failed: `log_warn "Failed to load LaunchAgent ${label}" && log_warn "Try manually: launchctl load $1"`

Internal function `_install_plist()`:
- Args: `$1` = template path, `$2` = destination plist filename
- `local dest="${HOME}/Library/LaunchAgents/${2}"`
- `ensure_directory "${HOME}/Library/LaunchAgents"`
- Copy template to destination: `cp "$1" "$dest"`
- Validate with plutil: `if ! plutil -lint "$dest" >/dev/null 2>&1; then die "Invalid plist: ${dest}" "Check template: ${1}"; fi`
- log_info "Installed plist: ${dest}"

Exported function `_install_launch_agents()`:
- Called by phase_9_complete (or directly by install.sh)
- Copy helper scripts to install dir:
  ```bash
  sudo cp "${SCRIPT_DIR}/scripts/backup.sh" "${AGMIND_DIR}/scripts/backup.sh"
  sudo cp "${SCRIPT_DIR}/scripts/health-gen.sh" "${AGMIND_DIR}/scripts/health-gen.sh"
  sudo chown "$(whoami)" "${AGMIND_DIR}/scripts/backup.sh" "${AGMIND_DIR}/scripts/health-gen.sh"
  chmod +x "${AGMIND_DIR}/scripts/backup.sh" "${AGMIND_DIR}/scripts/health-gen.sh"
  ```
- Ensure scripts and logs dirs exist:
  ```bash
  ensure_directory "${AGMIND_DIR}/scripts"
  ensure_directory "${AGMIND_DIR}/logs"
  ```
- Install backup plist: `_install_plist "${SCRIPT_DIR}/templates/launchd/com.agmind.backup.plist.template" "com.agmind.backup.plist"`
- Install health plist: `_install_plist "${SCRIPT_DIR}/templates/launchd/com.agmind.health.plist.template" "com.agmind.health.plist"`
- Load both: `_load_launch_agent "${HOME}/Library/LaunchAgents/com.agmind.backup.plist"`
- `_load_launch_agent "${HOME}/Library/LaunchAgents/com.agmind.health.plist"`
- log_info "LaunchAgents installed and loaded"

Note: `SCRIPT_DIR` is defined in install.sh and available to all sourced lib modules. Use `${SCRIPT_DIR:-$(cd "$(dirname "${BASH_SOURCE[0]}")/.." && pwd)}` as a fallback when sourced independently.
